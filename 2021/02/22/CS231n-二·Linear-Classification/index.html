Linear Classifier

и®ҫж•°жҚ®йӣҶдёә $\{(x_i,y_i)\mid i=1,2,\ldots,N\}$пјҢе…¶дёӯ $x_i\in\mathbb R^D,\,y_i\in\{1,\ldots,K\}$.

е®ҡд№ү score function: $f:\mathbb R^D\mapsto \mathbb R^K$пјҢеҚіиҫ“е…ҘдёҖдёӘз”ұеӣҫзүҮеғҸзҙ жһ„жҲҗзҡ„ $D$ з»ҙеҗ‘йҮҸпјҢиҫ“еҮәиҝҷдёӘеӣҫзүҮеұһдәҺеҗ„дёӘзұ»еҲ«зҡ„ score. жҲ‘д»¬и®Өдёә score и¶Ҡй«ҳпјҢеӣҫзүҮи¶ҠеҸҜиғҪеұһдәҺиҝҷдёӘзұ»еҲ«гҖӮ

еҜ№дәҺ linear classifier жқҘиҜҙпјҢе®ғзҡ„ score function еҫҲз®ҖеҚ•пјҡ

е…¶дёӯпјҢ$W,b$ жҳҜеҸӮж•°пјҢеҲҶеҲ«з§°дҪң weights е’Ң bias. жҳҫз„¶пјҢиҝҷйҮҢ $W\in\mathbb R^{K\times D},\,b\in\mathbb R^K$.

еҰӮжһңеҚ•зӢ¬зңӢз¬¬ $i$ зұ»зҡ„еҫ—еҲҶпјҢдёә $W$ зҡ„з¬¬ $i$ иЎҢдёҺ $x$ зҡ„еҶ…з§ҜеҠ дёҠ $b$ зҡ„з¬¬ $i$ дёӘе…ғзҙ гҖӮеҸҜд»ҘеҸ‘зҺ°зұ»еҲ«д№Ӣй—ҙзҡ„еҫ—еҲҶзӣёдә’зӢ¬з«ӢпјҢеҶҷжҲҗзҹ©йҳөеҸӘжҳҜдёәдәҶж–№дҫҝиҖҢе·ІгҖӮ


дёәдәҶзҗҶи§Ј linear classifierпјҢжіЁж„Ҹ $z=wx+b$ жҳҜдёҖдёӘ $\mathbb R^{D+1}$ дёӯзҡ„и¶…е№ійқўпјҢжі•еҗ‘йҮҸдёә $(w,-1)$пјҢжҲӘи·қдёә $b$. иҫ“е…Ҙзҡ„ $x$ еңЁи¶…е№ійқўдёҠеҜ№еә”дҪҚзҪ®зҡ„вҖңй«ҳеәҰвҖқи¶Ҡй«ҳпјҢе°ұи¶ҠеҸҜиғҪеұһдәҺиҝҷдёҖи¶…е№ійқўд»ЈиЎЁзҡ„йӮЈдёҖзұ»гҖӮдёӢйқўжҳҜ $D=2$ зҡ„дёҖдёӘдҫӢеӯҗпјҢзәҝжқЎиЎЁзӨәи¶…е№ійқўдёҺ $z=0$ зҡ„дәӨзәҝпјҢжІҝзқҖз®ӯеӨҙж–№еҗ‘иө°пјҢ$wx+b$ еҸҳеӨ§гҖӮ


$W,b$ жҳҜжҲ‘д»¬иҰҒи®ӯз»ғзҡ„еҸӮж•°пјҢдёәдәҶи®ӯз»ғе®ғд»¬пјҢжҲ‘д»¬йңҖиҰҒе®ҡд№ү Loss function. иҝҷж ·и®ӯз»ғ $W,b$ е°ұжҳҜжңҖе°ҸеҢ– Loss function зҡ„иҝҮзЁӢпјҢжңәеҷЁеӯҰд№ й—®йўҳжңҖз»ҲеҪ’з»“дёәдёҖдёӘдјҳеҢ–й—®йўҳгҖӮ

Multiclass SVM

дёҖз§Қ loss function жҳҜ Multiclass SVM Loss. еҜ№дәҺж•°жҚ® $(x_i,y_i)$ жқҘиҜҙпјҢе…¶ loss function е®ҡд№үдёәпјҡ

е…¶дёӯпјҢ$s_j$ иЎЁзӨә $x_i$ еңЁз¬¬ $j$ зұ»дёҠзҡ„еҫ—еҲҶпјҢеҚі $s_j=f(x_i;W,b)_j$.

еҜ№иҝҷдёӘ loss function зҡ„зҗҶи§ЈжҳҜпјҢеҰӮжһңжӯЈзЎ®еҲҶзұ»зҡ„еҫ—еҲҶ $s_{y_j}$ жҜ”й”ҷиҜҜеҲҶзұ»зҡ„еҫ—еҲҶ $s_j$ иҝҳй«ҳдёҖдёӘ $\Delta$пјҢйӮЈд№ҲжҚҹеӨұе°ұжҳҜ $0$пјӣеҗҰеҲҷпјҢжҚҹеӨұжҳҜ $s_j+\Delta$ жҜ” $s_{y_j}$ еӨҡеҮәзҡ„йғЁеҲҶгҖӮз”ұдәҺиҝҷдёӘеҮҪж•°зҡ„еӣҫеғҸеҪўзҠ¶еғҸй“°й“ҫпјҲеҗҲеҸ¶пјүпјҢжүҖд»Ҙиҝҷз§Қ loss function д№ҹз§°дҪң hinge loss.


еңЁжңәеҷЁеӯҰд№ дёӯеӯҰиҝҮзҡ„жӯЈеҲҷеҢ–д№ҹиҰҒз”ЁдёҠпјҢиӢҘдҪҝз”Ё $\text{L2 regularization}$пјҢеҲҷ Multiclass SVM Loss зҡ„е®Ңж•ҙеҪўејҸжҳҜпјҡ

еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢдёҠејҸдёӯзҡ„ $\Delta$ е№¶дёҚжҳҜйңҖиҰҒи°ғиҠӮзҡ„ hyperparameterпјҢеҸ– $\Delta=1$ еҚіеҸҜгҖӮиҝҷжҳҜеӣ дёә $W$ зҡ„ж•ҙдҪ“ж”ҫзј©еҸҜд»ҘеңЁдёҚж”№еҸҳ $s_j$ е’Ң $s_{y_i}$ зҡ„еӨ§е°Ҹе…ізі»зҡ„жқЎд»¶дёӢж”№еҸҳе®ғд»¬зҡ„е·®еҖјпјҢжүҖд»Ҙ $\Delta$ еҸ–еҖје№¶дёҚеҪұе“Қз»“жһңгҖӮ
