读入

  • 返回一个 reader 对象，可以迭代遍历 csvfile 中的所有行。csvfile 可以是任何支持 iterator 的对象，如文件或者列表。如果 csvfile 是文件对象，打开文件时需要指定 newline=''（详见下例）。

  • 可选项 dialect 是编码风格，默认是 excel 风格，分隔符为 ,（逗号）。

    可选项 fmtparams 用来覆盖当前 dialect 的格式参数。

    例如，参数位置写上 delimiter = '|' 就可以将分隔符指定为 | 。

    又如，doublequote，quotechar 等控制引号问题。

    更多内容见：Dialects and Formatting Parameters

  • 从 csv 文件读取的每一行返回为一个字符串列表。

写入

  • 返回一个 writer 对象，将用户数据转换为带分隔符的字符串。csvfile 可以是任何支持 write() 方法的对象。如果 csvfile 是文件对象，打开文件时需要指定 newline=''（详见下例）。

  • 可选项 dialect 是编码风格，默认是 excel 风格，分隔符为 ,（逗号）。

    可选项 fmtparams，用来覆盖当前 dialect 的格式参数。和读入一样。

  • 非字符串将通过 str() 被转换为字符串类型写入；空字符串将写作 None 写入。

  • writer 对象含有的方法：

    • writerow() 方法：写入一行（参数是一个列表）
    • writerows() 方法：写入多行（参数是一个列表的列表）

按字典方式读入

  • 创建一个类似 reader 的对象，但是它将每一行映射为一个字典，字典的键由 fieldnames 决定。
  • 可选参数 fieldnames 是一个 sequence。如果省略，文件 f 中的第一行将被认为是字典的键。
  • 如果某一行含有多于 fieldnames 的信息，剩余的信息将被放进一个列表，映射到键为 restkey 处，restkey 默认为 None。
  • 如果某一行含有少于 fieldnames 的信息，缺失的信息将被填充为 restval，默认为 None。

A short usage example:

按字典方式写入

  • 创建一个类似 writer 的对象，但是它将每一行映射为一个字典。
  • 参数 fieldnames 是一个 sequence。注意这里它不是可选参数。
  • restval 同读入。
  • 可选参数 extrasaction 可以是 raise 或 ignore，当某一行字典含有不在 fieldnames 中定义的键时，它用来决定是否抛出异常 ValueError。

Reader 对象

reader 和 DictReader 对象有以下方法：

  • csvreader.__next__()

    返回 reader 的下一行，以列表（如果是 reader() ）或者字典的形式（如果是 DictReader() ）

    通常以 next(reader) 的形式调用

reader 对象有以下属性：

  • csvreader.line_num

    行数

  • csvreader.dialect

    当前编码风格的只读描述

DictReader 对象有以下属性：

  • csvreader.fieldnames:

    如果在创建对象时没有传参，默认取读入的第一条记录为 fieldnames。

Writer 对象

writer 和 DictWriter 对象有以下方法：

  • csvwriter.writerow(row)

    按照当前编码风格写入一行

  • csvwriter.writerows(rows)

    按照当前编码风格写入多行

writer 对象有以下属性：

  • csvwriter.dialect

    当前编码风格的只读描述

DictWriter 对象有以下方法：

  • DictWriter.writeheader()

    将 field names 按照当前编码风格写入文件
